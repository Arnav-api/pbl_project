E-Commerce Website with AI-Powered Damage Detection

This project presents a web-based intelligent system designed to automate the product listing process using deep learning techniques. The system leverages Convolutional Neural Networks (CNN) for object classification and YOLOv8 for real-time damage detection.

By analyzing uploaded product images, the system identifies the product category and evaluates its condition, enabling users to generate accurate and reliable listings with minimal manual effort. This improves efficiency and enhances trust in e-commerce platforms.

Overview

This presentation outlines the development of an AI-powered e-commerce system, covering key aspects such as problem identification, literature review, system design, methodology, and performance evaluation.

The primary goal is to demonstrate how integrating computer vision techniques into web applications can significantly improve the efficiency and accuracy of product listing systems.

Introduction

Traditional e-commerce platforms rely heavily on manual input for product listings, requiring users to provide details such as category, description, and condition. This process is time-consuming and prone to human errors, leading to inconsistencies in listings.

Our system introduces an AI-driven approach where users simply upload an image, and the system automatically identifies the product and evaluates its condition, thereby reducing manual effort and improving accuracy.

Problem Statement

Existing e-commerce platforms lack automation in product listing, resulting in inefficient workflows and inconsistent data quality. Users often misclassify products or provide incorrect condition details, which reduces trust in online marketplaces.

Additionally, most current systems do not integrate object classification and damage detection, creating a gap in automated product evaluation. This highlights the need for a unified AI-based solution.

Literature Review

Advances in computer vision and deep learning have significantly improved image classification and object detection capabilities. Convolutional Neural Networks (CNNs) are widely used for feature extraction and classification tasks, while YOLOv8 provides high-speed and accurate real-time object detection.

However, most existing research focuses on either classification or damage detection separately. There is limited work on integrated systems that combine both functionalities within a web-based application.

System Methodology

The system workflow begins with the user uploading a product image. The image is then preprocessed, including resizing and normalization, to prepare it for model input.

A CNN model classifies the product into categories such as car, phone, or watch. Following this, YOLOv8 is applied to detect any visible damage. The results are displayed to the user, and a complete product listing is generated automatically.

Technologies Used

The system utilizes CNN models for accurate image classification due to their ability to capture spatial features effectively. YOLOv8 is used for damage detection because of its real-time performance and high detection accuracy.

For deployment, web frameworks such as Flask or Django are used to integrate the machine learning models into a user-friendly interface, making the system accessible and scalable.

Proposed Solution

The proposed system provides a fully automated pipeline where users upload an image, and the system performs both classification and damage detection simultaneously. Based on the results, a complete product listing is generated, including category and condition.

This approach reduces manual effort, improves consistency, and enhances user experience, making it highly suitable for modern e-commerce platforms.

AI System

The AI system integrates CNN and YOLOv8 models to deliver accurate and efficient performance. It can identify multiple product categories such as cars, watches, and phones, while also determining whether the product is damaged or undamaged.

This combination ensures reliable product classification and condition assessment, reducing human errors and improving trust in online marketplaces.

Model Performance & Evaluation

The performance of our AI system was evaluated using standard object detection metrics such as Precision, Recall, mAP@50, and mAP@50-95. These metrics provide a comprehensive understanding of the model’s detection accuracy, reliability, and generalization capability.

Precision indicates how many detected objects were actually correct, while Recall measures how effectively the model identifies all relevant objects. The mAP@50 and mAP@50-95 metrics evaluate the model across different Intersection over Union (IoU) thresholds, ensuring robustness.
